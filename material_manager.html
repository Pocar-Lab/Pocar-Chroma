Manager for materials used in photon tracking simulations.

This class reads material data from a CSV file, initializes Material objects, and manages material properties for simulation purposes.

get_material(self, material_name)[source]¶

Retrieve a Material object by its name.

Notes

  • This class relies on the ‘Material’ class from the chroma.geometry module.

  • Material properties are initialized from the CSV file and can be modified or updated during simulation.

  • The CSV file should contain columns for ‘name’, ‘refractive_index’, ‘absorption_length’, ‘scattering_length’, and ‘density’ for each material.
